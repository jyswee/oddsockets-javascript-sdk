OddSockets Round-Trip Demo

This page performs a full pub/sub round-trip against the OddSockets platform: connect → subscribe to a unique channel → publish a message → receive your own message back. It targets the manager endpoint https://connect.oddsockets.tyga.network.

Your key is only used in this page and never stored. You can also set window.ODDSOCKETS_API_KEY before this script runs and it will be picked up automatically.
